# Imágenes del sitio AWAY

Coloca aquí tus imágenes con **exactamente** estos nombres (minúsculas):

| Archivo         | Dónde se usa                                  | Tamaño recomendado        |
|-----------------|-----------------------------------------------|---------------------------|
| `logo.png`      | Esquina superior izquierda (navbar) y footer  | 64×64 px (cuadrado)       |
| `fondo.png`     | Imagen de portada grande en la página inicial | 1920×1080 px (panorámica) |
| `imagen1.png`   | Galería (1)                                   | 1200×900 px               |
| `imagen2.png`   | Galería (2)                                   | 1200×900 px               |
| `imagen3.png`   | Galería (3)                                   | 1200×900 px               |

> Si alguna imagen no existe, la página sigue funcionando: el hueco se
> muestra vacío o con un fondo oscuro, pero no hay errores.

## Notas

- El `logo.png` también se usa como favicon del navegador.
- `fondo.png` se muestra a pantalla completa en la home, así que cuanto
  más grande y nítida mejor. Si pesa mucho, optimízala (PNG/WebP).
- Las imágenes de la galería se ven mejor con relación de aspecto 4:3.
